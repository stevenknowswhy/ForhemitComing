§ 02State Filing Checklist
☐Articles of Conversion — Filed with {{stateSecretary}}. Fee: {{stateFee}}. Processing time: {{processingTime}}.
☐Articles of Incorporation — New C-Corp articles filed simultaneously with conversion. Include: name, registered agent, authorized shares, incorporator.
☐Bylaws — Draft C-Corp bylaws. Must include: board structure, officer roles, meeting requirements, ESOP provisions.
☐Initial board resolution — Officers elected, bank accounts authorized, ESOP authorized.
☐Share issuance — Issue shares to the seller (pre-ESOP). Shares will be sold to the ESOP trust at closing.
☐EIN update — Apply for new EIN if entity type change requires it (check IRS rules). Update with IRS using Form 8822-B.
☐State tax registration — Update registration with state tax authority. Some states require new registration for entity change.
☐Business licenses — Update all business licenses, permits, and registrations to reflect new entity name/type.
☐Bank accounts — Update authorized signers and entity name with all banks. New signature cards required.
☐Insurance policies — Update named insured on all policies (GL, D&O, E&O, property, workers' comp).
☐Contracts — Notify all material contract counterparties of entity change. Assign or novate contracts as needed.
§ 03Tax Election Checklist
☐Check-the-box election — If converting from LLC (taxed as partnership or disregarded entity), file Form 8832 to elect C-Corp treatment. Timing: must be effective before ESOP transaction closes.
☐S-Corp revocation — If currently an S-Corp, file Form 1128 to revoke the S election. Revocation must be effective before ESOP transaction. Note: S-Corp revocation triggers a 5-year waiting period for §1042 eligibility — confirm timing with CPA.
☐Final partnership return — If converting from partnership/LLP, file final Form 1065 for the period ending on the conversion date.
☐Built-in gains — If converting from S-Corp to C-Corp, the 5-year built-in gains tax applies to appreciated assets. CPA should model the exposure.
☐State tax elections — Some states require separate entity classification elections. Verify state conformity with federal election.
