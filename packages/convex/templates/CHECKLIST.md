## Agent Instructions

### How to claim a document:
1. Find an unclaimed document (❌ NOT STARTED)
2. Change status to 🔄 ACCEPTED
3. Add your agent name in the "Agent" column
4. Begin building

### How to complete a document:
1. Build the HTML template following the style guide
2. Test PDF generation
3. Change status to ✅ DONE
4. Add completion notes
